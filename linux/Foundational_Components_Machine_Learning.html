3.8. Machine Learning¶

Jacinto Machine Learning toolkit brings machine learning to the edge by enabling machine learning inference on all Jacinto devices. It is provided as part of TI’s Processor SDK Linux, free to download and use. Jacinto machine learning today consists of Neo-AI-DLR library.
